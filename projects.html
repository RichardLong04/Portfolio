MONKAA

Monkaa, was one of my first project at Seattle University. The objective was to embed media into a website using HTML and CSS. I was provided all of the media assets and images. I decided to make everything nice and centered, so it would be easier to see quickly. I made the site responsive, even though it wasn’t required. I wanted to challenge myself and learn more.

challenges

  • layout of each section
  • getting video to load quickly
  • responsive mobile first design

solutions

  • watched many tutorials on flex box and css grid
  • compressed the video for quicker loading

ISUD

ISUD, this was my final project for Spring term. This by far was one of the most challenging learning experiences to date for me. The objective was to redesign/improve a poorly designed existing website, as a team working remotely utilizing a GitHub group repository. We would meet every few days on meetup to discuss the project and see what each team member was working on. My biggest take away from this project was learning that having clear, concise goals and communication will be key to success with any future projects.

challenges

  • communication with in a group
  • Git Hub group repeositories and commits
  • streamlining three people's CSS to make a cohesive site

solutions

  • patience and listening while communicating with group
  • set clear roles for each group member
  • sought help with merging files when needed

TECHNICAL MARKETING INC..

TMI, was the second site I did for a customer. My client wanted his existing site to be updated with new content and given a modern make over. I used Bootstrap 3 and used the layout of the old site as my reference for the new site. Also, the old site wasn’t responsive and that was something the client wanted too. I used images from the old site and cleaned them up in photoshop, and I had all of their company’s clients send me current .png files of their new logos to work into a logo parade for the home page. In the end the customer was happy and I got paid.

challenges

  • creating responsive grid layouts
  • pleasing a client
  • getiing content to line up perfectly on mobile version

solutions

  • I learned how to use columns and rows that were colapsable
  • took many notes when being told talking with my client
  • many hours of trial and falure until everything worked perfectly
